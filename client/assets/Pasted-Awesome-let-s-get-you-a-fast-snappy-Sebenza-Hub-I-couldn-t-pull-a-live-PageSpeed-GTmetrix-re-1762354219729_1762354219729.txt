# 30–60 minute quick wins (highest ROI)

1. **Turn on Brotli + HTTP/2 (or HTTP/3) + CDN cache**

* Put the site behind **Cloudflare / Bunny / Fastly**. Cache static assets aggressively and enable Brotli.
* Why: reduces transfer size, parallelises requests globally. (GTmetrix/Lighthouse grade benefits.) ([gtmetrix.com][2])

2. **Compress & modernise images**

* Convert hero/illustrations to **WebP/AVIF**; set responsive `srcset` and `sizes`; add `loading="lazy"` to below-the-fold media.
* Aim: <100 KB per hero where possible; thumbnails <20–40 KB.

3. **Kill render-blocking**

* Add `defer` to non-critical scripts; load analytics after `load` or via `defer`.
* Inline only the minimal critical CSS; push the rest async.

4. **Fonts: self-host Montserrat**

* Preload the exact files you use (e.g., 400, 600) and set `font-display: swap`.
* Remove unused weights/styles.

5. **Ship less JavaScript**

* Code-split routes/components (`React.lazy`, `Suspense`), tree-shake, and remove heavy UI libs you don’t need.
* Target <200–300 KB JS (gzipped) on first load for best LCP/TBT.

# Express (Node) settings for instant gains

```js
// app.js
import express from 'express';
import compression from 'compression';
import helmet from 'helmet';

const app = express();

// 1) Security headers (minor perf from better caching control)
app.use(helmet({
  contentSecurityPolicy: false, // set CSP separately if needed
}));

// 2) Brotli/Gzip
app.use(compression());

// 3) Far-future caching for static assets with immutable hashes
app.use('/assets', express.static('dist/assets', {
  setHeaders: (res, path) => {
    if (/\.(js|css|woff2|png|jpg|jpeg|webp|avif)$/i.test(path)) {
      res.setHeader('Cache-Control', 'public, max-age=31536000, immutable');
    } else {
      res.setHeader('Cache-Control', 'public, max-age=3600');
    }
  }
}));

// 4) Minimal JSON: strip whitespace, paginate, only send what’s needed
app.set('json spaces', 0);

export default app;
```

# Build pipeline

* Use **Vite** (or Webpack) with:

  * **Minify** (Terser/Esbuild), **splitChunks**, **manualChunks** (vendor split).
  * **Bundle analysis** (`source-map-explorer` or `rollup-plugin-visualizer`) to delete dead weight.
* Strip **moment locales**, unused icons, and duplicate CSS.
* Prefer **Tailwind** (JIT) or utility CSS over large component suites.

# Database/API layer (keeps TTFB low)

* Add **indexes** to hot columns, paginate everywhere, and cache read-heavy endpoints (Redis or in-process LRU).
* Compress JSON responses (already covered), and avoid N+1 queries.
* Return lean payloads: only fields you render above the fold.

# “Copy-paste” CDN rules (Cloudflare)

* **Auto Minify**: JS/CSS/HTML (Dashboard → Speed).
* **Brotli**: On.
* **Caching**:

  * Page Rule / Cache Rule: `/assets/*` → Cache Everything, Edge TTL 1 year.
  * Bypass cache for `/api/*` and authenticated pages.
* **Early Hints / HTTP/3**: On.

# Common gotchas (worth checking)

* **Heavy hero video or carousel** on homepage: convert to short muted MP4/WebM + lazy poster; or swap to a static hero.
* **Google Fonts FOUT/FOIT**: self-host + preload (above).
* **3rd-party scripts** (chat widgets, A/B tools): load after interaction; remove what you don’t use.
* **Layout shift**: always set `width`/`height` (or aspect-ratio) on images/iframes to stabilise CLS.
